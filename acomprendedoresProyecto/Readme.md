# Acomprendedores - Sistema bancario

## Uso:
El proyecto consiste en un sistema bancario con tres perfiles: admin, con funcionalidades de gestion de elementos como productos financieros, usuarios, algunas estadisticas; el cliente puede hacer transacciones (depositos, retiros, etc.); el gestor de ventas puede consultar productos financieros de un usuario especifico o gestionarle un prestamo. Se planteo como el incio de aprendizaje de C# y del sistema bancario basico.

## Tecnologias utilizadas
- **Lenguaje:** C#
- **Framework:** Windows Forms, .NET
- **Base de datos:** SQL Server
